Embedded GUI: Widgets Within Editors

Mike T. McHenry

 (require embedded-gui)

The embedded-gui library provides a class hierarchy for creating graphical
boxes within editor<%> objects with geometry management that mirrors that of
vertical-panel% and horizontal-panel%.
