12 Platform Dependencies

See Platform Dependencies in The Racket Drawing Toolkit for information on
platform library dependencies for racket/draw. On Unix, the following
additional system libraries must be installed for racket/gui/base:

  • "libgdk-x11-2.0[.0]"

  • "libgdk_pixbuf-2.0[.0]"

  • "libgtk-x11-2.0[.0]"

  • "libgdkglext-x11-1.0[.0]" — optional, for OpenGL support

  • "libgtkglext-x11-1.0[.0]" — optional, for OpenGL support

  • "libunique-1.0[.0]" — optional, for single-instance support
